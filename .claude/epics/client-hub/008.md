# Task: Dashboard Page and Layout

## Description

Create a professional dashboard layout with sidebar navigation and header, plus a dashboard home page that serves as the landing page after agent login. Implements modern UI/UX following mobile-first design principles.

## Design Specifications

**Design System:**
- Primary Color: #2563EB (Trust Blue)
- Secondary Color: #3B82F6
- CTA Color: #F97316 (Orange)
- Background: #F8FAFC
- Text: #1E293B
- Border: #E2E8F0
- Typography: Poppins (headings), Open Sans (body)

**Style:**
- Minimalist Swiss Style with clean grids
- Mobile-first responsive design (375px+)
- WCAG AAA accessibility compliance
- Smooth transitions (200-300ms)
